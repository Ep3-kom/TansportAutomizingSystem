# TAS — Setup & Installatie

## Vereisten

- **Node.js** (v18 of hoger) — [https://nodejs.org](https://nodejs.org)
- **Git** — [https://git-scm.com](https://git-scm.com)

Controleer of je alles hebt:

```bash
node -v
npm -v
git --version
```

---



---

## 1. Frontend installeren

```bash
cd frontend
npm install
```

---

## 2. App starten

```bash
npm run dev
```

De app draait nu op **http://localhost:5173**

---

## 3. App stoppen

Druk op `Ctrl + C` in de terminal.

---

## Samenvatting (copy-paste dit)

```bash
cd TansportAutomizingSystem/frontend
npm install
npm run dev
```

Klaar! Open **http://localhost:5173** in je browser.
